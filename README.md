>A simple react Js project where i'm use [Breaking Bad API](https://breakingbadapi.com/documentation) to show and filter characters


# Technology Used:

1-React JS
## Quick Start:

```bash
# Install dependencies
npm install

# Run App
npm start

# Build for production
npm run build
```
